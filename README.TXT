实验器材:
	战舰STM32F103开发板V3版本
	
实验目的:
	学习STM32驱动OV7670\OV7725等摄像头.
	
硬件资源:
	1,DS0(连接在PB5)
	2,串口1(波特率:115200,PA9/PA10连接在板载USB转串口芯片CH340上面)
	3,ALIENTEK 2.8/3.5/4.3/7寸TFTLCD模块(通过FSMC驱动,FSMC_NE4接LCD片选/A10接RS)
	4,按键KEY0(PE4)/KEY1(PE3)/KEY2(PE2)/KEY_UP(PA0,也称之为WK_UP)
	5,TPAD电容触摸按键(右下角LOGO,即TPAD,连接在PA1)
	6,外部中断8(PA8,用于检测OV7670\OV7725的帧信号)
	7,定时器6(用于打印摄像头帧率)
	8,ALIENTEK OV7670摄像头模块,连接关系为:
		OV7670\OV7725模块 ----------- STM32开发板
		 OV_D0~D7  -----------   PC0~7
		 OV_SCL    ------------  PD3
		 OV_SDA    ------------  PG13
		 OV_VSYNC  ------------  PA8
		 FIFO_RRST -----------   PG14
		 FIFO_OE   -----------   PG15
		 FIFO_WRST ------------  PD6
		 FIFO_WEN  ------------  PB3
		 FIFO_RCLK ------------  PB4	  
		 
实验现象:
	本实验开机后，先初始化摄像头模块，OV7670初始化成功，则在LCD模块上面显示摄像头模块所
	拍摄到的内容;OV7725初始化成功，则在需要先按KEY0\KEY1选择OV7725为QVGA\VGA模式输出。
	我们可以通过KEY0设置光照模式（5种模式）、通过KEY1设置色饱和度，通过KEY2设置亮度，通过
	WK_UP设置对比度，通过TPAD设置特效（总共7种特效）。通过串口，我们可以查看当前的帧率
	（这里是指LCD显示的帧率，而不是指OV7670\OV7725的输出帧率），DS0指示程序运行状态。
	同时可以借助USMART设置OV7670\OV7725的寄存器，方便大家调试。调用SCCB_RD_Reg和SCCB_WR_Reg
	等函数,实现OV7670\OV7725寄存器读写.
	
	
	OV7725_QVGA模式：传感器输出230*240分辨率输出至LCD，可视角度广，但画面没那么清晰和细腻。
	OV7725_VGA模式：将640*480分辨率截取中间的320*240输出至LCD，可视角度窄，但画面清晰和细腻。
